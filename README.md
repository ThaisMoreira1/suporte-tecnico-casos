# 📌 Suporte Técnico - Casos Práticos

Este repositório apresenta exemplos de situações reais de suporte técnico, com foco em diagnóstico, resolução de problemas e boas práticas baseadas em ITIL.

---

## 🧩 Casos de Suporte

### 🔹 Sistema lento
**Cenário:** Usuário relata lentidão no sistema  
**Ação:**
- Coleta de informações (horário, frequência, impacto)
- Verificação de uso de CPU, memória e rede
- Análise de possíveis gargalos  
**Solução:** Identificação de sobrecarga e ajuste no ambiente

---

### 🔹 Erro de login
**Cenário:** Usuário não consegue acessar o sistema  
**Ação:**
- Validação de credenciais
- Teste de redefinição de senha
- Verificação de status do serviço de autenticação  
**Solução:** Correção de falha no login/autenticação

---

### 🔹 Sistema indisponível
**Cenário:** Sistema fora do ar  
**Ação:**
- Verificação de servidores e serviços
- Consulta de incidentes globais
- Comunicação com o cliente  
**Solução:** Restabelecimento do serviço e monitoramento
